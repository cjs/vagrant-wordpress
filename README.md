# Setting up a Wordpress host #

## Requirements ##
    Virtualbox => https://www.virtualbox.org/wiki/Downloads
    Vagrant    => http://downloads.vagrantup.com

## Setup ##
    vagrant up
    vagrant provision

## Using Wordpress ##
    your wordpress site will be available via http://localhost:8080.
